# 增强功能
1. 记录、导出查询历史
2. 自动朗读控制
3. 去除官方[重定向百度首页“后门”](http://wenku.baidu.com/view/e9660493aef8941ea76e05cf "后门报告")
4. 修复无法保存设置
5. 采用新技术（shadowDOM）避免与浏览页面样式冲突（v3.1.6)
6. 持续的优化

# Distribute
```
npm run dist
```
